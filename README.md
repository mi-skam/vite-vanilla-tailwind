# Template: Vite+TailwindCSS

## Install

```bash
npx degit mi-skam/vite-vanilla-tailwind your-project

# or simply clone this repo with git clone mi-skam/vite-vanilla-tailwind
```

## Commands

```bash
# install
npm install

# develop
npm run dev

# build-out
npm run build

# preview, needs build-out
npm run serve
```
